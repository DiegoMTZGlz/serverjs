# Servidor HTTP en JS

Ejercicio sobre la ejecución de un servidor HTTP con JS

## Para iniciar ¿Qué es un servidor HTTP?

Un servidor HTTP es un programa informático que procesa una aplicación del lado del servidor, realizando conexiones bidireccionales o unidireccionales y síncronas o asíncronas con el cliente y generando o cediendo una respuesta en cualquier lenguaje o aplicación del lado del cliente.

### Prerequisitos de ejecución

Node-js
Términal

### Instalación para la ejecución

Para instalar Node-js deberemos ejecutar los siguientes comandos:

```
$ sudo apt install nodejs
```

Para verificar que la instalación se haya hecho correctamente usamos:

```
$ node -v
```

Si nos arroja una versión de node-js significa que dicha versión ha sido instalada en nuestro sistema.

### Ejecución de la práctica

Una vez naveguemos hasta donde se encuentren los archivos correspondientes a la práctica, tan solo deberemos ejecutar el comando:

```
node etapa<#>.js
```
De esta manera se ejecutará el servidor de la etapa correspondiente remplanzando "#" por valores del 1 al 5.
